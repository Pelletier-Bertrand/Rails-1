#Projet : Petit blog des familles

Une application de blog. C'est un super blog où ses utilisateurs peuvent créer des articles, les commenter, leur attribuer à une catégorie et liker les articles.


##Les modèles :

* Les users

* Les articles

* Les catégories

* Les commentaires

* Les likes
